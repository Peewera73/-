หน่วยที่ 4 การเขียนรหัสเทียม

ความหมายของรหัสเทียม
คือ คือ รหัสเทียม (Pseudo Code) หรือ ซูโดโค้ด เป็นคําสั่งที่ไม่ขึ้นกับภาษาคอมพิวเตอร์ภาษาใดภาษาหนึ่ง แต่เป็นคําสั่งที่นํามาใช้เขียนเลียนแบบชุดคําสั่งแบบย่อ ๆ เพื่อออกแบบโปรแกรม หรือร่างเค้าโครงโปรแกรมที่ จะเขียนขึ้นมา ก่อนที่จะนําไปแปลงเขียนเป็นชุดคําสั่งด้วยภาษาคอมพิวเตอร์จริง ๆ

การเขียนรหัสเทียม (Pseudo Code) นําไปใช้ประโยชน์เพื่อเป็นพื้นฐานในการเขียนโปรแกรมดังนี้
  1. นํามาใช้ทบทวนความถูกต้องกับสิ่งที่ได้ออกแบบไป

  2. นํามาใช้เพื่อกําหนดงานเขียนโปรแกรม เพื่อให้โปรแกรมเมอร์นําไปเพิ่มเติมรายละเอียดและใช้เป็นแนวทางในการเขียนโปรแกรมต่อไป

  3. นํามาใช้เป็นเอกสารประกอบโปรแกรม เพื่องานบํารุงรักษาโปรแกรมที่อาจเกิดขึ้นในอนาคต

  4. นํามาใช้เป็นเครื่องมือเพื่อการออกแบบต่าง ๆ ที่เกี่ยวข้องกับเทคนิคเชิงโครงสร้าง เช่น การออกแบบโมดูลโปรแกรม การออกแบบโปรแกรมแบบบนลงล่าง เป็นต้น

หลักการเขียนรหัสเทียม
คือ การออกแบบขั้นตอนวิธีหรืออัลกอริทึมเพื่อแก้ไขปัญหาใดปัญหาหนึ่ง ขั้นตอนแรก ต้องเข้าใจใน เบื้องต้นก่อนว่า คอมพิวเตอร์นั้นจะปฏิบัติงานตามชุดคําสั่งที่เขียนตามความเป็นจริง ดังนั้น หากมีการกําหนด ถ้อยคําหรือประโยคในรูปแบบของรหัสเทียม ต้องมีความสอดคล้องกับการปฏิบัติงานทางคอมพิวเตอร์ ขั้นพื้นฐานเพื่อให้การแปลงชุดคําสั่งจากรหัสเทียมมาเป็นภาษาคอมพิวเตอร์ได้ง่ายยิ่งขึ้นการเขียนรหัสเทียมไม่มีการกําหนดมาตรฐานที่ระบุไว้ชัดเจน ดังนั้น ผู้เขียนรหัสเทียมแต่ละคน อาจมี เทคนิคการเขียนที่แตกต่างกัน แต่ทั้งนี้เพื่อให้การเขียนรหัสเทียมมีแบบแผนขึ้นจึงควรปฏิบัติตามต่อไปนี้

  1. คือ ถ้อยคําต่าง ๆ ให้เขียนเป็นภาษาอังกฤษอย่างง่าย

  2. คือ แต่ละคําสั่งให้เขียนอยู่ในแต่ละบรรทัด

  3. คือ ควรใช้ย่อหน้าให้เป็นประโยชน์และแยกคําเฉพาะให้เห็นชัดเจน นอกจากนี้ควรจัดรูปแบบโครงสร้าง ควบคุมให้เป็นสัดส่วน เพื่อให้อ่านง่าย

  4. คือ เขียนประโยคคําสั่งเรียงลําดับแบบบนลงล่าง มีทางเข้าและทางออกเพียงทางเดียวเท่านั้น

  5. คือ กลุ่มประโยคคําสั่งต่าง ๆ อาจถูกนํามาจัดรวมกลุ่มเข้าด้วยกันในรูปแบบของโมดูล และทําการ กําหนดชื่อโมดูลขึ้นมา เพื่อให้ส่วนโปรแกรมหลักหรือโมดูลย่อยอื่น ๆ เรียกใช้งานได้

การตั้งชื่อตัวแปรให้สื่อความหมาย
คือ ชุดคําสั่งของรหัสเทียมจะมีการกําหนดและอ้างอิงใช้ตัวแปร ต่าง ๆ ซึ่งโปรแกรมเมอร์นอกจากต้องกําหนดชื่อตัวแปรไม่ให้ซ้ํากันแล้ว ยังควรตั้งชื่อตัวแปรให้สื่อถึง ความหมาย เพื่อประโยชน์ต่อการอ้างอิงใช้งานและสามารถตีความได้ทันทีว่าตัวแปรนี้เก็บค่าใด ตัวอย่างเช่น ตัวแปรที่ใช้เก็บชื่อนักศึกษา การตั้งชื่อตัวแปรว่า std_name ย่อมสื่อความได้ดีกว่า n หรือชื่อตัวแปรที่เก็บค่า ตัวเลข 3 ค่า การตั้งชื่อว่า number1, number2 และ number3 ย่อมสามารถสื่อความได้ดีกว่า a, b และ c นอกจากนี้หากเขียนติดกันจะทําให้สะกดคํายาก จึงใช้เครื่องหมาย _ (Underscore) หรือใช้ชื่อตัวพิมพ์ใหญ่ ช่วยแยกคําเพื่อให้อ่านง่ายขึ้น เช่น sales_tax หรือ salesTax เป็นต้น

การประยุกต์ใช้รหัสเทียม
คือ การนํารหัสเทียมไปประยุกต์ใช้เพื่อปฏิบัติการคอมพิวเตอร์ขั้นพื้นฐาน เพื่อมุ่งเน้นการเขียนประโยค ถ้อยคําเป็นชุดคําสั่งภาษาอังกฤษแบบตรงไปตรงมา ทําให้ผู้อ่านเข้าใจง่าย รวมถึงการจัดย่อหน้าและควบคุม โครงสร้างให้เป็นสัดส่วน โดยการประยุกต์ใช้รหัสเทียมเพื่อปฏิบัติการคอมพิวเตอร์ขั้นพื้นฐาน ประกอบด้วย

  1. การประยุกต์ใช้รหัสเทียมเพื่อการรับข้อมูล คือ คอมพิวเตอร์สามารถรับข้อมูลจากแหล่งใดแหล่งหนึ่งได้ไม่ว่าจะเป็นเทอร์มินัล ดิสก์แป้นพิมพ์ หรืออุปกรณ์อื่น ๆ โดยคําสั่ง READ, GET และ INPUT จะถูกนํามาใช้งานในลักษณะดังนี้ READ ปกติจะถูกนํามาใช้เมื่อมีการอ่านเรคอร์ดข้อมูลจากแฟ้มข้อมูล (อาจใช้ควบคู่กับคําสั่ง OPEN ที่ใช้สําหรับเปิดแฟ้มข้อมูล) GET, INPUT เป็นคําสั่งที่มักถูกนํามาใช้กับกรณีการรับข้อมูลจากแป้นพิมพ์

  2. การประยุกต์ใช้รหัสเทียมเพื่อการนําส่งข้อมูลไปยังอุปกรณ์ต่าง ๆ คือ เมื่อคอมพิวเตอร์ต้องการนําส่งข้อมูลเพื่อแสดงผลไปยังอุปกรณ์ภายนอก คําสั่งที่ใช้คือ PRINT, WRITE, PUT, OUTPUT หรือ DISPLAY โดยที่ PRINT ปกติจะถูกนํามาใช้เมื่อมีการส่งข้อมูลออกไปยังเครื่องพิมพ์ WRITE นํามาใช้เมื่อมีการส่งข้อมูลออกไปเพื่อบันทึกลงในไฟล์หรือแฟ้มข้อมูล PUT, OUTPUT หรือ DISPLAY ถูกนํามาใช้เมื่อมีการนําข้อมูลออกทางจอภาพ

  3. การประยุกต์ใช้รหัสเทียมเพื่อการคํานวณ คือ โปรแกรมที่เขียนขึ้นโดยส่วนใหญ่ มักให้คอมพิวเตอร์คํานวณหรือจัดเรียงข้อมูล โดยเฉพาะสูตร คํานวณต่าง ๆ โปรแกรมเมอร์สามารถใช้สัญลักษณ์สูตรทางคณิตศาสตร์จริง ๆ มาใช้หรืออาจใช้ข้อความแทน สัญลักษณ์ก็ได้เช่น ADD number to total หรือ total = total + number ซึ่งมีความหมายเดียวกัน

  4. การประยุกต์ใช้รหัสเทียมเพื่อการกําหนดค่าตัวแปร คือ การเขียนรหัสเทียมเพื่อกําหนดค่าให้กับตัวแปรมี 3 กรณีซึ่งประกอบด้วย 1. เขียนรหัสเทียมเพื่อกําหนดค่าเริ่มต้นให้กับตัวแปร โดยใช้คํากริยา INITIALISE หรือ SET 2. กําหนดค่าที่ได้จากการประมวลผลบางอย่าง โดยใช้สัญลักษณ์ = หรือ ← 3. เก็บค่าตัวแปรไว้ใช้งานภายหลัง โดยใช้คํากริยา SAVE หรือ STORE

  5. การประยุกต์ใช้รหัสเทียมเพื่อการเปรียบเทียบค่าและกําหนดทางเลือกตามเงื่อนไขคือ การประยุกต์ใช้รหัสเทียมที่สําคัญอีกอย่างคือ ความสามารถในการเปรียบเทียบค่าสองค่า ซึ่ง ผลลัพธ์จากการเปรียบเทียบ จะมีเพียงหนึ่งตัวเลือกที่ได้รับการดําเนินตามกิจกรรมที่ตรงตามเงื่อนไขนั้น ๆ โดย รหัสเทียมที่นํามาใช้เพื่อการเปรียบเทียบค่าคือ IF–THEN–ELSE โดยมีคําสั่ง ENDIF ปิดท้ายบล็อกที่ใช้สร้าง เงื่อนไขดังกล่าวทุกครั้ง โดยที IF แสดงถึงประโยคเริ่มต้นที่จะทําการกําหนดเงื่อนไข คําสั่งที่อยู่ถัดจาก THEN คือ กิจกรรมที่จะได้รับการดําเนินงาน หากเงื่อนไขเป็นจริง คําสั่งที่อยู่ถัดจาก ELSE คือ กิจกรรมที่จะได้รับการดําเนินงาน หากเงื่อนไขเป็นเท็จ

  6. การประยุกต์ใช้รหัสเทียมเพื่อการทํางานซ้ำ ๆ คือ การทํางานในบางครั้ง จําเป็นต้องใช้วิธีวนซ้ำเพื่อประมวลผลกิจกรรมซ้ำ ๆ ภายในลูปจํานวน หลายรอบ เช่น กรณีรับข้อมูลนักศึกษา จํานวน 50 คน จะต้องใช้ลูปเพื่อรับข้อมูลนักศึกษา ซ้ำ ๆ กัน 50 รอบ โดยการเขียนรหัสเทียมเพื่อสร้างลูปทําซ้ำกัน จะใช้คําเฉพาะต่อไปนี้ WHILE…ENDWHILE เป็นโครงสร้างการวนซ้ำโดยจะตรวจสอบเงื่อนไขก่อนประมวลผล ภายในลูป REPEAT…UNTIL เป็นโครงสร้างการวนซ้ำโดยจะมีการประมวลผลชุดคําสั่งภายในลูปอย่าง น้อยหนึ่งรอบเสมอ แล้วจึงค่อยตรวจสอบเงื่อนไขการวนซ้ำ FOR…NEXT เป็นโครงสร้างการวนซ้ำที่มีการกําหนดจํานวนรอบการทํางานที่แน่นอน

สรุปหน่วยที่ 4

รหัสเทียม (Pseudo Code) หรือ ซูโดโค๊ด
คือ คําสั่งที่ไม่ขึ้นกับภาษาคอมพิวเตอร์ภาษาใดภาษาหนึ่ง แต่ เป็นคําสั่งที่นํามาใช้เขียนเลียนแบบชุดคําสั่งแบบย่อ ๆ เพื่อออกแบบโปรแกรม หรือร่างเค้าโครงโปรแกรมที่จะเขียน ขึ้นมา ก่อนที่จะนําไปแปลงเขียนเป็นชุดคําสั่งด้วยภาษาคอมพิวเตอร์จริง ๆ ต่อไป โดยสามารถนําไปใช้ประโยชน์ เพื่อเป็นพื้นฐานในการเขียนโปรแกรมดังนี้

  1. นํามาใช้ทบทวนความถูกต้องกับสิ่งที่ได้ออกแบบไป

  2. นํามาใช้เพื่อกําหนดงานเขียนโปรแกรม เพื่อให้โปรแกรมเมอร์นําไปเพิ่มเติมรายละเอียดและใช้เป็น แนวทางในการเขียนโปรแกรมต่อไป

  3. นํามาใช้เป็นเอกสารประกอบโปรแกรม เพื่องานบํารุงรักษาโปรแกรมที่อาจเกิดขึ้นในอนาคต

  4. นํามาใช้เป็นเครื่องมือเพื่อการออกแบบต่าง ๆ ที่เกี่ยวข้องกับเทคนิคเชิงโครงสร้าง

หลักการเขียนรหัสเทียม ประกอบด้วย
  1. ถ้อยคําต่าง ๆ ให้เขียนเป็นภาษาอังกฤษอย่างง่าย

  2. แต่ละคําสั่งให้เขียนอยู่ในแต่ละบรรทัด

  3. ควรใช้ย่อหน้าให้เป็นประโยชน์และแยกคําเฉพาะให้เห็นชัดเจน นอกจากนี้ควรจัดรูปแบบโครงสร้าง ควบคุมให้เป็นสัดส่วน เพื่อให้อ่านง่าย

  4. เขียนประโยคคําสั่งเรียงลําดับแบบบนลงล่าง มีทางเข้าและทางออกเพียงทางเดียวเท่านั้น

  5. กลุ่มประโยคคําสั่งต่าง ๆ อาจถูกนํามาจัดรวมกลุ่มเข้าด้วยกันในรูปแบบของโมดูล

การประยุกต์ใช้รหัสเทียม เพื่อปฏิบัติการคอมพิวเตอร์ขั้นพื้นฐาน เป็นการเรียนรู้การใช้คําสั่ง เพื่อนํามา ประยุกต์ใช้กับการเขียนรหัสเทียม ซึ่งประกอบด้วย
  1. การประยุกต์ใช้รหัสเทียมเพื่อการรับข้อมูล ประกอบด้วย คําสั่ง READ, GET และ INPUT

  2. การประยุกต์ใช้รหัสเทียมเพื่อการนําส่งข้อมูลไปยังอุปกรณ์ต่าง ๆ ประกอบด้วย คําสั่ง PRINT, WRITE, PUT, OUTPUT และ DISPLAY

  3. การประยุกต์ใช้รหัสเทียมเพื่อการคํานวณ ประกอบด้วย คําสั่ง ADD, SUBTRACT, MULTIPLY, DIVIDE และ PARENTHESES

  4. การประยุกต์ใช้รหัสเทียมเพื่อการกําหนดค่าตัวแปร ประกอบด้วย คําสั่ง INITIALISE, SET, SAVE, STORE และสัญลักษณ์ =, ←

  5. การประยุกต์ใช้รหัสเทียมเพื่อการเปรียบเทียบค่า และกําหนดทางเลือกตามเงื่อนไข ประกอบด้วย คําสั่ง IF–THEN–ELSE โดยจะมีคําสั่ง ENDIF ปิด
