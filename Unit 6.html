หน่วยที่ 6 การทำงานแบบตัดสินใจ

ความหมายของการทำงานแบบตัดสินใจ
คือ การทํางานแบบตัดสินใจจะมีการทํางานที่มีความซับซ้อนกว่าการทํางานแบบตามลําดับ โดยใช้ในการ ตรวจสอบเงื่อนไขทางตรรกะว่าเป็นจริงหรือเท็จ หากผลตรวจสอบมีค่าเป็นจริงก็จะไปทําคําสั่งหนึ่ง ถ้าเป็นเท็จ ก็จะไปทําอีกคําสั่งหนึ่ง

การทํางานแบบตัดสินใจด้วย if–statement
คือ การทํางานแบบตัดสินใจด้วย if–statement เป็นการตรวจสอบเงื่อนไขทางตรรกะว่าเป็นจริงหรือเท็จ โดยประกอบด้วย 4 รูปแบบ

  1. การสร้างเงื่อนไข if อย่างง่าย คือ การตรวจสอบเงื่อนไขว่าเป็นจริงหรือเท็จ ถ้าเป็นจริงแล้วให้ทํา ชุดคําสั่งภายใต้เงื่อนไขนั้น ถ้าเป็นเท็จทําคําสั่งถัดไป



    2. ตัวอย่าง1.1ตรวจสอบเงื่อนไขของตัวแปรชื่อ var ว่ามีค่าเท่ากับ 0 หรือไม่ ถ้าเท่ากับ 0 ให้แสดง ข้อความว่า “Devide by Zero”



    ตัวอย่าง1.2ตรวจสอบเงื่อนไขของตัวแปร score ว่ามีค่าตั้งแต่ 80 หรือไม่ถ้าใช่ ให้พิมพ์ข้อความ แสดงความยินดีว่า “Excellent !!!”



  2. การสร้างเงื่อนไข if–elseคือ การสร้างเงื่อนไขเพื่อรองรับเหตุการณ์ทั้งจริงและเท็จ กล่าวคือ หากเงื่อนไขที่ตรวจสอบมีค่าเป็นจริงก็จะทําตามคําสั่งหลัง if แต่ถ้าเป็นเท็จก็จะทําตามคําสั่งหลัง else



    3. ตัวอย่าง2.1แสดงการตรวจสอบเงื่อนไข if–else หรือการตรวจสอบเงื่อนไขสองทางเลือก โดยหากเงื่อนไขเป็นจริงจะดําเนินการตามชุดคําสั่งหนึ่ง หากเงื่อนไขเป็นเท็จก็จะดําเนินการตามอีกชุดคําสั่ง หนึ่ง



    ตัวอย่าง2.2ตรวจสอบเงื่อนไข ค่าตัวเลขจํานวนเต็มที่ป้อนเข้ามาว่าเป็นเลขคู่ หรือ เลขคี่ ถ้าเป็นเลข คู่แสดงข้อความ “Even number…” ถ้าเป็นเลขคี่แสดงข้อความว่า “Odd number…”



    ตัวอย่าง2.3ตรวจสอบเงื่อนไขคะแนนที่รับเข้ามาทางแป้นพิมพ์หากได้คะแนนตั้งแต่ 50 คะแนนขึ้นไป ให้แสดงข้อความว่า “PASS” ถ้าไม่ใช่แสดงข้อความ



  3. การสร้างเงื่อนไข if–else แบบหลายกรณี คือ รูปแบบเงื่อนไขเพียงแค่ 2 กรณีเท่านั้น ดังนั้น หาก รูปแบบการสร้างเงื่อนไขที่ต้องการตรวจสอบมีหลายกรณีจะใช้ประโยค else if เพื่อตรวจสอบเป็นลําดับขั้น ย่อย ๆ ต่อไป

    4. ตัวอย่าง3.1ตรวจสอบเงื่อนไขของคะแนน เพื่อหาเกรดที่ได้



  4. การสร้างเงื่อนไขแบบซ้อน Nested ifคือ รูปแบบการสร้างเงื่อนไขที่มีความซับซ้อนยิ่งขึ้น โดย จะมีการตรวจสอบเงื่อนไขซ้อนย่อยลงไปอีก ซึ่งการสร้างประโยคซ้อนเงื่อนไขดังกล่าว จําเป็นต้องตรวจให้ รอบคอบ มิฉะนั้นอาจเกิดผลลัพธ์ที่ผิดพลาดได้

    5. ตัวอย่าง4.1ตรวจสอบเงื่อนไขของ salary ว่าอยู่ใน rate ใด



การทํางานแบบตัดสินใจด้วย switch…case
คือ รูปแบบคําสั่งการตัดสินใจแบบ switch…case เป็นคําสั่งที่ช่วยในการเขียนโปรแกรมให้มีความกระชับ มากขึ้น ใช้ในกรณีที่มีคําสั่ง IF…THEN…ELSE…IF (Nested IF) จํานวนมากเพื่อช่วยทําให้โปรแกรมอ่านง่าย สามารถตรวจสอบและแก้ไขโปรแกรมได้ง่าย

การประยุกต์ใช้การทํางานแบบตัดสินใจ
คือ การประยุกต์การทํางานแบบตัดสินใจในหน่วยนี้จะยกตัวอย่างกรณีศึกษาการเขียนโปรแกรม คอมพิวเตอร์ที่มีการทํางานแบบตัดสินใจ เพื่อแสดงให้เห็นความสัมพันธ์ของแต่ละขั้นตอน ตั้งแต่ การกําหนด และวิเคราะห์ปัญหา การเขียนผังงาน การเขียนรหัสเทียม และการเขียนโปรแกรม ตามลําดับ
