หน่วยที่ 10 การเขียนโปรแกรมแบบตามลำดับและแบบตัดสินใจ

การเขียนโปรแกรมการทํางานแบบตามลําดับ
ลักษณะการเขียนโปรแกรมการทํางานแบบตามลําดับต้องมีในทุกโปรแกรมคอมพิวเตอร์โปรแกรมจะ ทํางานตามคําสั่งที่เขียนไว้ตามลําดับ ตั้งแต่คําสั่งแรกไปจนถึงคําสั่งสุดท้าย

การเขียนโปรแกรมการทํางานแบบ ตัดสินใจด้วย if–statement
ภาษาซีมีการทํางานแบบตัดสินใจด้วย if–statement เพื่อตรวจสอบเงื่อนไขทางตรรกะว่าเป็นจริงหรือ เท็จ หากผลตรวจสอบมีค่าเป็นจริงจะไปทําคําสั่งหนึ่ง หรือเป็นเท็จจะไปทําอีกคําสั่งหนึ่ง นอกจากนี้การทํางาน แบบตัดสินใจด้วย if–statement มีหลายรูปแบบด้วยกัน

  1. การเขียนโปรแกรมการทํางานแบบมีเงื่อนไข if อย่างง่าย การเขียนโปรแกรมการทํางานแบบมีเงื่อนไข if อย่างง่ายเป็นการตรวจสอบเงื่อนไขว่าเป็นจริง หรือเท็จ ถ้าเป็นจริงแล้วให้ทําชุดคําสั่งภายใต้เงื่อนไขนั้น ถ้าเป็นเท็จทําคําสั่งถัดไป

  2. การเขียนโปรแกรมการทํางานแบบมีเงื่อนไข if–elseการเขียนโปรแกรมการทํางานแบบมีเงื่อนไข if–else เป็นการสร้างเงื่อนไขเพื่อรองรับ เหตุการณ์ทั้งจริงและเท็จ กล่าวคือ หากเงื่อนไขที่ตรวจสอบมีค่าเป็นจริงจะทําตามคําสั่งหลัง if แต่ถ้าเป็นเท็จ จะทําตามคําสั่งหลัง else



  3. การเขียนโปรแกรมการทํางานแบบมีเงื่อนไข if–else แบบหลายกรณี รูปแบบเงื่อนไข if–else ข้างต้น จะเป็นรูปแบบเงื่อนไขเพียงแค่ 2 กรณีเท่านั้น ดังนั้น หาก รูปแบบการสร้างเงื่อนไขที่ต้องการตรวจสอบหลาย ๆ กรณีก็จะใช้ประโยค else if เพื่อตรวจสอบเป็นลําดับขั้น ย่อย ๆ ต่อไป



  4. การเขียนโปรแกรมการทํางานแบบมีเงื่อนไขแบบซ้อน Nested if การเขียนโปรแกรมการทํางานแบบมีเงื่อนไขแบบซ้อน Nested if เป็นรูปแบบการสร้าง เงื่อนไขที่มีความซับซ้อนยิ่งขึ้น โดยจะมีการตรวจสอบเงื่อนไขซ้อนย่อยลงไปอีก ซึ่งการสร้างประโยคซ้อน เงื่อนไขดังกล่าว จําเป็นต้องตรวจให้รอบคอบ เพราะมิฉะนั้นอาจเกิดผลลัพธ์ที่ผิดพลาดได้



    5. การเขียนโปรแกรมการทํางานแบบตัดสินใจด้วย switch…case
    รูปแบบคําสั่งการตัดสินใจแบบ switch…case เป็นคําสั่งที่ช่วยในการเขียนโปรแกรมให้มีความกระชับ มากขึ้น ใช้ในกรณีที่มีจํานวนคําสั่ง IF…THEN…ELSE…IF (Nested if) เพื่อช่วยทําให้โปรแกรมอ่านง่าย สามารถ ตรวจสอบและแก้ไขโปรแกรมได้ง่าย

การเขียนโปรแกรมการทํางานแบบตามลําดับ
แสดงถึงการเขียนโปรแกรมด้วยคอมพิวเตอร์ที่มีการ ทํางานแบบตามลําดับ จะทํางานตามคําสั่งแรกไปจนถึงคําสั่งสุดท้าย

การเขียนโปรแกรมการทํางานแบบตัดสินใจด้วย if–statement
ลักษณะการเขียนโปรแกรมด้วย if–statement นํามาใช้เพื่อตรวจสอบเงื่อนไขทางตรรกะว่าเป็นจริงหรือเท็จ ซึ่งมีรูปแบบการเขียนโปรแกรมใน ลักษณะต่าง ๆ ดังนี้

  1. การเขียนโปรแกรมแบบมีเงื่อนไข if แบบทางเลือกเดียว

  2. การเขียนโปรแกรมแบบมีเงื่อนไข if–else

  3. การเขียนโปรแกรมแบบมีเงื่อนไข if–else แบบหลายกรณี

  4. การเขียนโปรแกรมแบบมีเงื่อนไขแบบซ้อน (Nested if)

    5. การเขียนโปรแกรมการทํางานแบบตัดสินใจด้วย switch…case
    ลักษณะการเขียนโปรแกรมแบบ switch…case นํามาใช้งานได้ดีกับโปรแกรมที่มีรายการเมนูให้เลือกและไม่สามารถนํามาใช้ตรวจสอบเงื่อนไข ด้วยนิพจน์ที่เป็นเลขจํานวนจริงและมีจุดทศนิยมได้
